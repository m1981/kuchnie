## 🪵 PLAYBOOK #09: MONTAŻ BLATU HPL (KRYTYCZNY! 3-4 godz.)

**CEL:** Dociąć i zamontować wodoodporny blat kompaktowy bez pęknięć.

### ⚠️ ZASADY FUNDAMENTALNE (przeczytaj 3 RAZY!)

```
🔴 REGUŁA #1: Blat HPL to NIE płyta wiórowa!
   → Rdzeń z żywicy fenolowej - twardy jak kamień
   → Zwykła tarcza do drewna go NIE przetnie (spali się!)

🔴 REGUŁA #2: Nigdy OSTRYCH KĄTÓW 90°!
   → Każdy narożnik otworu (zlew, płyta) = ZAOKRĄGLONY!
   → Promień min. 5mm (wiertło 10mm w narożnikach)

🔴 REGUŁA #3: Blat musi "oddychać"!
   → NIE przykręcać na sztywno!
   → Tylko klej elastyczny punktowo

🔴 REGUŁA #4: Transportuj TYLKO W PIONIE!
   → Poziomo się zarysuje od siebie
```

### 🔧 KROK 1: Narzędzia i materiały (przygotuj PRZED rozpoczęciem)

```
NARZĘDZIA:
☐ Zagłębiarka + szyna prowadząca (najważniejsze!)
☐ Tarcza HM do laminatów - 60+ zębów (np. Bosch Expert)
☐ Frezarka + frez 6mm, 8mm
☐ Wiertarka + wiertła: 5mm, 8mm, 10mm
☐ Wiertło Forstner 35mm (jeśli zlew wpuszczany)
☐ Otwornica 68mm (pod baterie jeśli montuje)
☐ Dalmierz laserowy (pomiary precyzyjne)
☐ Kątownik stalowy 30cm
☐ Miara zwijana 5m
☐ Poziomica 60cm
☐ Ołówek + gumka
☐ Taśma malarska (ochrona dekoru przy cięciu)
☐ Odkurzacz (kurz z cięcia HPL jest WSZĘDZIE!)

MATERIAŁY:
☐ Blat HPL 12mm (docięty do szerokości ± zapas)
☐ Klej Ottocoll M500 (2 tuby)
☐ Osłona termiczna samoprzylepna (pod płytę indukcyjną)
☐ Silikon bezbarwny (uszczelnienie styku ze ścianą)
☐ Ściereczki mikrofibr (czyszczenie)
☐ Papier ścierny 120 (wygładzenie krawędzi)

INSTRUKCJE AGD (KONIECZNIE!):
☐ Instrukcja montażu zlewozmywaka (wymiary wycięcia!)
☐ Instrukcja montażu płyty indukcyjnej (wymiary wycięcia!)
```

### 📏 KROK 2: Aklimatyzacja blatu (30 min przed cięciem)

```
☐ Jeśli blat był w zimnym magazynie / aucie:
   ☐ Wnieś do pomieszczenia klienta
   ☐ Połóż poziomo na szafkach dolnych (już ustawionych!)
   ☐ Odczekaj 30 minut do temp. pokojowej 18-25°C
   
⚠️ DLACZEGO? Zimny blat jest bardziej kruchy - może pęknąć przy cięciu!
```

### ✂️ KROK 3: Pomiar i oznaczenie cięć (PRECYZJA!)

```
☐ Pomiar DŁUGOŚCI blatu:
   ☐ Zmierz szerokość szafek dolnych (cała linia)
   ☐ Dodaj występ z przodu: +3-4 cm (standard)
   ☐ Dodaj zapas z tyłu: +0,5 cm (zabezpieczenie przy ścianie)
   ☐ PRZYKŁAD: Szafki 250cm + występ 4cm + tył 0,5cm = 254,5 cm
   
☐ Pomiar GŁĘBOKOŚCI blatu:
   ☐ Standardowa głębokość: 60 cm (szafki 60cm)
   ☐ Jeśli szafki płytsze (np. 55cm bo rury): dostosuj!

☐ Oznaczenie linii cięcia NA SPODZIE blatu:
   ☐ Połóż blat SPODEM DO GÓRY na szafkach / stole
   ☐ Odmierz wymiary dalmierzem
   ☐ Zaznacz linię OŁÓWKIEM (dobrze widoczna!)
   ☐ Naklej TAŚMĘ MALARSKĄ wzdłuż linii cięcia
      → Chroni dekor przed wyszczerbieniem

🔴 ZASADA CIĘCIA HPL:
   → ZAWSZE tnij DEKOREM DO GÓRY! (piękna strona górą)
   → Tarcza wchodzi od góry - wyrzuca wióry w dół (spód)
   → Dzięki temu dekór pozostaje idealnie gładki
```

### 🪚 KROK 4: Cięcie blatu zagłębiarką

```
☐ Montaż tarczy HM do laminatów (60+ zębów):
   ☐ Sprawdź kierunek montażu (strzałka na tarczy!)
   ☐ Dokręć mocno klucz

☐ Ustawienie głębokości cięcia:
   ☐ Głębokość = grubość blatu + 2mm
   ☐ Dla blatu 12mm → ustaw 14mm

☐ Montaż szyny prowadzącej:
   ☐ Ustaw szynę równolegle do linii cięcia
   ☐ Sprawdź odległość: od krawędzi podstawy zagłębiarki do brzeszczotu
   ☐ Przykręć szynę za pomocą klamer (lub użyj ciężarów)

☐ CIĘCIE (WOLNO I SPOKOJNIE!):
   ☐ Włącz odkurzacz (podłącz do zagłębiarki)
   ☐ Włącz zagłębiarkę - niech nabierze obrotów (10 sek)
   ☐ Prowadź WOLNO wzdłuż szyny (nie forsuj!)
   ☐ Nie zatrzymuj się w połowie cięcia (równomierny ruch)
   ☐ Na końcu cięcia - lekko zwolnij (unikniesz wyrwania)

☐ Sprawdzenie cięcia:
   ☐ Oderwij taśmę malarską
   ☐ Sprawdź czy krawędź gładka (bez wyszczerbień)
   ☐ Jeśli drobne nierówności: papier ścierny 120
```

## 🧊 PLAYBOOK #10: MONTAŻ AGD - LODÓWKA W ZABUDOWIE

**Źródło:** Bazuje na `SOP-vertical/08-Montaż-AGD.md` + rozszerzenie

**CEL:** Bezkolizyjne osadzenie lodówki, zgranie fugi frontów, wentylacja.

### 🔴 ZASADY FUNDAMENTALNE

```
1. Lodówka MUSI oddychać - kratka wentylacyjna koniecznie!
2. Fronty lodówki = zawiasy 155° (ZERO USKOKU!)
3. Suwaki/zawieszki montuj ROZEBRANE (nie na gotowo)
4. Test "na sucho" PRZED finalnym montażem
```

---

### 📏 KROK 1: Weryfikacja wymiarów (test "na sucho")

```
☐ Otwórz instrukcję lodówki (KONIECZNIE!):
   ☐ Sprawdź RZECZYWISTE wymiary (nie katalogowe!)
   ☐ Przykład: "Wysokość 177cm, szerokość 54cm, głębokość 55cm"

☐ Problem: Osiadanie pod ciężarem
   → Lodówka w kartoniku ma 177,5cm
   → Po ustawieniu na nóżkach i załadowaniu = 176,8cm!
   → Różnica 7mm to katastrofa jeśli fronty nie pasują!

☐ TEST "NA SUCHO" (przed przykręceniem frontów!):
   
   1. Wsuń lodówkę do słupka (bez frontów meblowych)
   2. Zmierz odległość od podłogi do szparybetween górne/dolne drzwi lodówki:
      ☐ Zmierzona wysokość fugi lodówki: ____ mm (np. 1182 mm)
   
   3. Zmierz gdzie będzie fuga między Twoimi frontami MDF:
      ☐ Planowana fuga frontów: ____ mm (np. 1185 mm)
   
   4. PORÓWNAJ:
      ☐ Czy różnica <3mm? → OK, możesz montować
      ☐ Czy różnica >5mm? → PROBLEM!

☐ Jeśli fronty nie pasują do fugi lodówki:
   
   ROZWIĄZANIE: Podłóż lodówkę formatkami!
   ☐ Wysuń lodówkę
   ☐ Przytnij kawałki HDF/OSB/płyty 18mm na wymiar dna słupka
   ☐ Połóż formatki na dno
   ☐ Wsuń lodówkę ponownie
   ☐ Sprawdź czy teraz fugi się zgadzają (test 2)
   ☐ Reguluj dokładnie (1-2mm ma znaczenie!)
```

### 🔩 KROK 2: Usztywnianie słupka lodówkowego

```
Problem: Słupek bez pleców + ciężka lodówka = "galareta"

☐ Montaż szafki przed wsunięciem lodówki:
   ☐ Słupek złożony (z Playbook #08)
   ☐ Ustawiony w linii szafek dolnych
   ☐ JESZCZE NIE przykręcony na sztywno do sąsiedniej szafki

☐ Wsuń lodówkę do słupka:
   ☐ Powoli (2 osoby!)
   ☐ Sprawdź czy kabel zasilający nie jest przygnieciony

☐ Przykręcenie słupka do sąsiednich szafek:
```
