### 1. Wzorzec "Composite" (Kompozyt) do budowy drzewa BOM

Obecnie Twój program generuje płaską listę. W CAD/CAM BOM to zawsze **drzewo**. Szafka (Assembly) składa się z podzespołów (Sub-assemblies, np. Szuflada), a te z części (Parts, np. Prowadnica, Płyta).

**Rozwiązanie w Pythonie:**
Stwórz strukturę drzewiastą. Dzięki temu, gdy dodasz "Szufladę Legrabox", ona sama "wie", że musi dodać do siebie dno, tył i prowadnice.

```python
class BOMNode:
    def get_cost(self): pass
    def get_items(self): pass

class Part(BOMNode):
    def __init__(self, name, cost, quantity=1):
        self.name = name
        self.cost = cost
        self.quantity = quantity

    def get_cost(self):
        return self.cost * self.quantity

class Assembly(BOMNode):
    def __init__(self, name):
        self.name = name
        self.children = []

    def add(self, child: BOMNode):
        self.children.append(child)

    def get_cost(self):
        return sum(child.get_cost() for child in self.children)

# Użycie:
drawer = Assembly("Blum Legrabox 500mm")
drawer.add(Part("Prowadnice komplet", 45.00))
drawer.add(Part("Dno z płyty 16mm", 5.00))
drawer.add(Part("Wkręty 4x16", 0.50, quantity=12))

cabinet = Assembly("Drawer Base 800")
cabinet.add(drawer) # Dodajesz szufladę, a ona ciągnie za sobą resztę!
```

### 3. Wzorzec "Strategy" (Strategia) dla logiki zakupowej (Purchasing Logic)

To rozwiąże Twój problem z liczeniem płyt na m2 vs całe arkusze oraz blatów na metry bieżące.
Oddziel **BOM Netto** (to co fizycznie jest w szafce) od **BOM Brutto/Zakupowego** (to co musisz kupić).

```python
from math import ceil

class PurchasingStrategy:
    def calculate_purchase_qty(self, net_qty): pass

class SheetMaterialStrategy(PurchasingStrategy):
    def __init__(self, sheet_size_m2):
        self.sheet_size = sheet_size_m2

    def calculate_purchase_qty(self, net_qty_m2):
        # Zawsze zaokrągla w górę do pełnych arkuszy
        sheets_needed = ceil(net_qty_m2 / self.sheet_size)
        return sheets_needed * self.sheet_size

class LinearMeterStrategy(PurchasingStrategy):
    def __init__(self, standard_length):
        self.standard_length = standard_length

    def calculate_purchase_qty(self, net_length):
        # Np. blaty kupowane w sztangach 4.1m
        pieces = ceil(net_length / self.standard_length)
        return pieces * self.standard_length

# Użycie:
egger_board_strategy = SheetMaterialStrategy(sheet_size_m2=5.8)
# Jeśli program wyliczy 7.5 m2, strategia zakupowa powie: musisz kupić 11.6 m2 (2 arkusze)
```

### Krok 2: Wzorzec "Composite" (Drzewo BOM)

**Problem:** Twoja metoda `calculate_cost` zwraca płaską listę `CostTraceLine`. Nie wiesz, czy dany kawałek płyty należy do korpusu, czy do szuflady.
**Rozwiązanie:** Tworzymy strukturę drzewiastą. Ponieważ używasz Reflex, te klasy muszą dziedziczyć po `BaseModel` (Pydantic), aby mogły być wysłane do frontendu.

```python
from pydantic import BaseModel

class BOMNode(BaseModel):
    name: str
    cost: float = 0.0

class BOMPart(BOMNode):
    material_id: int | None = None
    quantity_net: float
    unit: str
    unit_price: float

    def calculate(self):
        self.cost = self.quantity_net * self.unit_price

class BOMAssembly(BOMNode):
    children: list['BOMNode'] = []

    def calculate(self):
        for child in self.children:
            if isinstance(child, BOMAssembly) or isinstance(child, BOMPart):
                child.calculate()
        self.cost = sum(child.cost for child in self.children)
```

### Krok 3: System Cech (Tags) i Silnik Reguł (Rules Engine)

**Problem:** W `calculate_cost` masz hardkodowane: `if self.height_mm > 1600: hinges_per_door = 4`.
**Rozwiązanie:** Tworzymy osobną klasę `RulesEngine`, która czyta tagi z `recipes.json` i buduje drzewo BOM.

**Nowy plik `engine.py`:**

```python
class BOMGenerator:
    def __init__(self, cabinet: Cabinet, recipe: dict, defaults: ProjectDefaults):
        self.cabinet = cabinet
        self.recipe = recipe
        self.defaults = defaults

    def generate_tree(self) -> BOMAssembly:
        root = BOMAssembly(name=f"{self.cabinet.name} Assembly")

        # 1. Generowanie korpusu na podstawie wzorów z JSON
        corpus_m2 = eval_formula(self.recipe["formulas"]["corpus_m2"], self.cabinet)
        root.children.append(BOMPart(
            name="Corpus Board",
            quantity_net=corpus_m2,
            unit="m2",
            unit_price=self.defaults.corpus_mat.price_per_unit
        ))

        # 2. Silnik reguł na podstawie tagów
        tags = self.recipe.get("tags", [])

        if "is_base" in tags:
            root.children.append(BOMPart(name="Cabinet Legs", quantity_net=4, unit="pcs", unit_price=1.5))

        if self.cabinet.door_count > 0:
            hinges = 2 if self.cabinet.height_mm <= 900 else (3 if self.cabinet.height_mm <= 1600 else 4)
            root.children.append(BOMPart(
                name="Hinges",
                quantity_net=hinges * self.cabinet.door_count,
                unit="pcs",
                unit_price=self.defaults.hinge_sys.price_per_set
            ))

        root.calculate()
        return root
```

**Problem:** Twój `waste_factor` (1.20x) jest mnożony na poziomie pojedynczej szafki. W rzeczywistości stolarz kupuje płyty dla całego projektu naraz.

```python
class Project(SQLModel, table=True):
    # ... istniejące pola ...

    def calculate_total_cost(self) -> dict:
        # 1. Zbieramy wszystkie BOMPart ze wszystkich szafek
        all_parts = []
        for cab in self.cabinets:
            recipe = load_recipe(cab.recipe_id)
            bom_tree = BOMGenerator(cab, recipe, self.defaults).generate_tree()
            all_parts.extend(extract_all_parts(bom_tree))

        # 2. Agregujemy materiały (np. sumujemy wszystkie m2 białej płyty)
        aggregated_materials = aggregate_by_material_id(all_parts)

        # 3. Aplikujemy Strategie Zakupowe
        total_material_cost = 0
        for mat_id, net_qty in aggregated_materials.items():
            material = get_material(mat_id)

            if material.category == "Board":
                # Strategia: Pełne arkusze (np. 5.8 m2)
                gross_qty = math.ceil(net_qty / 5.8) * 5.8
            elif material.category == "Countertop":
                # Strategia: Sztangi 4.1m
                gross_qty = math.ceil(net_qty / 4.1) * 4.1
            else:
                # Strategia: Dokładna ilość + ryczałtowy waste (np. obrzeże)
                gross_qty = net_qty * self.waste_factor

            total_material_cost += gross_qty * material.price_per_unit

        return {"total_cost": total_material_cost, "aggregated_bom": aggregated_materials}
```
